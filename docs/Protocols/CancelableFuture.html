CancelableFuture

public protocol CancelableFuture

When the Callable is executed an instance of an object that implements this protocol will be passed to it as it’s only argument. The Callable closure can use this to monitor the isCanceled flag to see if it should stop execution by throwing the error ExecutorError.CallableCanceled.
